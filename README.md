A physics-based balancing game built with Pygame.

## How to Play

1. Install Python 3.x
2. Install requirements: `pip install -r requirements.txt`
3. Run the game: `python main.py`

## Controls

- **W / S**: Lean Bike
- **Space**: Nitro Boost
- **K / L / G**: Change Theme
